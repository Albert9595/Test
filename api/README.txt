1)Данный скрипт служит для парсинга сайта etherscan.io по API.
2)Запустите виртуальное окружение  sourse api_etherscan/bin/activate.
3)Установите библиотеки из requirements.txt.
4)В api_ethe.py находиться основной скрипт (Задайте собственный api_key).
5)В sql_api.py находиться sql запросы (Здайте собственный путь к sql).
6)В conversion.py находиться функции конверсии из 16 системе счисления числа в 10 и наоборот.
7)В db.sqlite3 находиться сама база данных.
8)На данный момент база данных заполнена.
9)Можно вызвать скрипт bd.delet_info() и все сотрется.
10)Если хотите посмотреть "Баланс который изменился больше остальных" вызовите скрипт get_greatest_value() (Он вернет адрес).
11)Если хотите актуализировать данные вызовите скрипт etherscan_api() ждите пока база заполнится, в начале выполните пункт 9.
